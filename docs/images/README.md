# Katooni MVP — Image prompts

**Style for all prompts:** editorial fashion campaign. Grainy 35mm film (ISO 800). Dark ink-black background, one hard light source from above, volumetric haze. Cool desaturated colors. The shoe is the subject, 3/4 angle, sole visible. Negative space. 1 in 4 shots is black & white. 1 in 8 shots has a single volt yellow (#D7FF1E) accent — a sole, a stitch, a heel tab.

Generate as PNG, long edge 1600px minimum.
